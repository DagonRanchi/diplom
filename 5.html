Задачи проекта

01

Разработка модуля онлайн-подачи заявок

Создание веб-интерфейса для дистанционной регистрации и подачи документов абитуриентами.

02

Хранение персональных данных

Организация защищенной базы данных для сбора и обработки информации в соответствии с законодательством.

03

Автоматизация формирования личных дел

Генерация стандартизированных печатных форм, заявлений и описей документов без ручного ввода.

04

Интеграция с НОБД

Настройка автоматического обмена данными с Национальной образовательной базой данных.

05

Разграничение прав доступа

Реализация ролевой модели безопасности для сотрудников приемной комиссии и администраторов системы.

Дипломный проект: Разработка клиент-серверной системы для верификации абитуриентов с самозаполнением данных и интеграцией с НОБД
