Материалы для дипломного проекта УМЗ

Тема: разработка корпоративного веб-портала для сотрудников корпорации «УМЗ» с интегрированной системой электронного документооборота.

После финального запуска проекта сюда можно добавить новые скриншоты:

- публичная страница портала;
- форма создания карточки;
- табличный реестр документов;
- карточка документа;
- архив и папки;
- чаты;
- пользователи и роли;
- настройки системы.
